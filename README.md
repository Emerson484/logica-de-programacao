# logica-de-programacao
📂 Estrutura do Repositório

O projeto está dividido em três pastas principais:

Estruturas de Repetição: Implementações focadas em controle de fluxo utilizando laços for, while , do-while , if e if else.

Função e Recursão: Exercícios sobre modularização de código, passagem de parâmetros e a aplicação de lógica recursiva para problemas complexos.

Arrays e Ordenação: Trabalhos com vetores e listas, incluindo a manipulação de coleções e algoritmos para ordenação de elementos.

🛠️ Tecnologias Utilizadas
C++

Dev-c++
